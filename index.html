Group Projects

Group Project 1 - eQuipt





This app has been designed to help out the LARPing community (Live Action Role Play). It allows users to digitize their game inventories and allow trading among players while also allowing admin users the ability to create in-game items. To view the app itself, find it on Heroku here: https://larp-equipt.herokuapp.com/

Group Project 2 - Prodigies





Prodigies was a collaborative construction with my fellow coding schoolmates. It's intended use was to provide a forum and links to online educational games for young students. The intention was that students, parents, or teachers would be able add game to the site and students could rate them on their quality.

Group Project 3 - Your Day Today





Your Day Today is a daily dashboard of information for you today! Simply create an account/log in and you will be presented with weather information as well as today's news, interesting events in history that happened on today's date, as well as the ability to add to-do's to your list.

Project 4 - Psychic Game





In this app, the user types tries to guess the letter the computer is thinking of, but watch out, you have a limited number of guesses! The app also keeps track of the user's wins and losses.

Project 5 - Crystals Collector Game





This app is a game in which the user clicks a particular gem which adds a random number for that session to a counter. The goal of the game is to click the correct gems to add up exactly to a predetermined total. Try it out!

Project 6 - Matrix Hangman Game





This app is basically a reimagined version of the classic game: Hangman. Guess letters one at a time, take too many guesses and you lose. Guess all the letters in the given word and you win!
